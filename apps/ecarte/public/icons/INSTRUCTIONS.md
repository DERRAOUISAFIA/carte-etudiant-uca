# Icônes PWA

Générez deux icônes PNG avec le logo UCA :

- **192x192px** → `192.png`
- **512x512px** → `512.png`

Recommandations :
- Fond : `#102447` (bleu UCA)
- Logo : blanc ou couleur UCA officielle
- Format : PNG avec transparence ou fond plein
- Outils : Figma, Canva, ou https://realfavicongenerator.net/

Une fois générées, placez-les dans ce dossier et supprimez ce fichier.
